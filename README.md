# Web Development Projects

This repository is a collection of various web development projects and exercises, ranging from basic HTML/CSS to advanced React and Node.js applications.

## 🚀 Projects Overview

| Directory | Description | Tech Stack |
| :--- | :--- | :--- |

| **WebDev4** | Backend Integration & Servers | Node.js, Express |
| **WebDev5** | API Development | Node.js |
| **WebDev6** | Fullstack Applications | JS, Express |
| **WebDev7** | Frontend Frameworks | React |

| **flask** | Basic RestAPIs |

## 🛠️ Getting Started

To explore any of these projects locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/KirtanDwivedi/WebDev.git
   ```

2. **Navigate to a project folder**:
   ```bash
   cd WebDev/WebDev4  # For example
   ```

3. **Install dependencies** (if a `package.json` exists):
   ```bash
   npm install
   ```

4. **Run the project**:
   - For Node.js: `node server.js` or `npm start`
   - For Vite/React: `npm run dev`
